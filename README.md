Follow these steps to get started with the Linktree Clone:

1. Fork the repository

2. Clone the repository:

```bash

```

3. Change into the poject directory:

```bash

```

4. Install project dependencies:

```bash
npm install
```

5. Install project dependencies:

```bash
npm run dev
```
